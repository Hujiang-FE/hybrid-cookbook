### Introduction

Hybrid交互技术是无线端技术领域的热门话题，本着务实的宗旨，本书将对Hybrid开发过程中的常见问题（俗称：坑）、相关知识进行梳理，试图总结出一份技术严谨的参考手册，亦或是最佳实践。

我们的目标：促进大前端和谐发展，提升无线端产品体验
