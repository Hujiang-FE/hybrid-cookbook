Introduction
=======

［Hybrid混合应用开发模式/交互技术］是无线端技术领域的热门话题，本着务实的宗旨，本书将对Hybrid开发过程中的常见问题（俗称：坑）、相关知识进行梳理，试图总结出一份技术风格较为严谨的参考手册，亦或是最佳实践，希望抛砖引玉。

我们的目标：促进大前端和谐发展，提升无线端产品体验
